### Значки
Доступны 3 вида значков:

1) ![discount](app/src/main/res/drawable/discount.png)
Скидка на блюдо

2) ![leaf](app/src/main/res/drawable/leaf.png)
Блюдо без мяса

3) ![pepper](app/src/main/res/drawable/pepper.png)
Острое блюдо

Блюдо может содержать каждое из значков
